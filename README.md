# TheBoozeReaction

Web Application written in React.js + backbone.js to utilize boozefinder API

## Dependencies

Install node.js. (We're using v0.12.5)
`npm install` to install JavaScript dependencies

## Run the tests

`npm test`

## Create a build

`npm run build`

## Serve the build

`cd build && python -m SimpleHTTPServer` (for now)
